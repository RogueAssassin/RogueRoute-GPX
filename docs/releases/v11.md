# RogueRoute GPX v11

This v11 package focuses on safer rebuilds and more reliable OSRM preparation.

## Key changes

- Public version label is `v11`.
- npm package metadata remains semver-compatible as `11.0.0`.
- Docker build stages repair executable permissions in `node_modules/.bin` before building.
- OSRM preprocessing defaults to low-memory-safe settings.
- First-time install and rebuild/update flows are detected and shown in the installer/rebuild logs.
- OSRM graph validation now fails incomplete builds instead of starting a partial graph.

## Recommended OSRM defaults

```bash
OSRM_THREADS=2
OSRM_SAFE_THREADS=1
OSRM_FORCE_SAFE_BUILDS=true
# Optional, for example:
# OSRM_DOCKER_MEMORY=48g
```

For very large extracts such as `us-latest.osm.pbf`, use:

```bash
OSRM_THREADS=1 OSRM_DOCKER_MEMORY=96g ./prepare-osrm.sh repair us-latest --force --yes
```
