# GitHub Desktop Guide

## Uploading a new release
1. Open GitHub Desktop.
2. Clone `RogueRoute-GPX` or add the local repo.
3. Replace the repo contents with the updated release files.
4. Review the changed files.
5. Enter a summary such as `Release v7.5.0`.
6. Commit to `main`.
7. Push origin.
8. Create a GitHub release tagged `v7.5.0` and upload the zip and `.user.js` assets.
